    • 体で覚えるVue.js - ディレクティブ編 〜 JSおくのほそ道 #023 - Qiita

    <style>
[v-cloak] {
  display: none;
}
</style>

<div id="app">
  <div v-cloak>{{ foo }}</div>
</div>

    リクエストデータをコンポーネントで扱う¶

    リクエストデータをコンポーネントで扱うには body-parser を組み込む。ここで例示されているサンプルはNuxt.jsのバージョンが古いので注意。POSTリクエストのbodyの他にもいろいろ参照可能。

    • Working with Body Request Params req\.body · Issue #24 · nuxt/nuxt.js

    Express のミドルウェアとして body-parser をセットする順序に注意。 以下のように API Routes の前に定義しないと正しく動作しなかった。

    server/index.js 

    app.set('port', port)

// Express middlewares (need to load before routes)
app.use(bodyParser.urlencoded({ extended: false }))
app.use(bodyParser.json())

// Import API Routes
app.use('/api', api)

    pages/*.vue 

    async asyncData ({ req }) {
  let body = {}
  if (req && req.body) {
    body = req.body
  }
  return { body }
},
